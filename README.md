# H Wallet

## UI 协作入口

当前海豚社区 App 的 UI 设计协作、仓库互传与审核说明，请优先查看 [`UI-COLLAB-README.md`](./UI-COLLAB-README.md)。

设计师、前端与负责人后续统一以仓库文件为协作依据，并严格按以下顺序推进页面审核：**钱包页 → 币种明细页 → 聊天交易卡片 → 赚币页**。未进入 `deliverables/ui-review/90-approved/` 的版本，不作为最终开发依据。

H Wallet is a React Native + Expo app powered by OKX Onchain OS.
